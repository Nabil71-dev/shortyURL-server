## Overview
It's a simple URL shortner project which I created to utilize my recent learnings and tricks
###### Technologies and methologies
* ExpressJs,
* JWT, (access token & refresh token)
* CRON-job, 
* Redis cache, 
* Seeding, 
* ORM (mongoose), 
* SLS function, 
* bcrypt,
